# rounds.vin — Product Documentation

## What rounds.vin is
  Transcript software for VIN (Veterinary Information Network). Operators upload recorded sessions, the AI pipeline produces a first-pass transcript with speaker labels and slide alignment, and a workflow walks the transcript through Copy Edit → Medical Review → Publish before it ships to the CMS.
